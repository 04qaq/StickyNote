# 便签管理器（StickyNote Manager）产品需求文档

| 属性 | 内容 |
|------|------|
| 项目名称 | 便签管理器（StickyNote Manager） |
| 文档版本 | v2.0 |
| 技术栈 | Qt 5.15 Widgets + CMake + C++17 |
| 预估工期 | 1～2 周 |
| 最终产物 | 独立可运行的 .exe/.app 桌面应用 |

---

## 一、产品概述

便签管理器是一款桌面端便签管理工具，支持便签的创建、富文本编辑、分类、搜索、置顶、颜色标记、卡片拖拽排序、多窗口独立便签、撤销/重做操作栈、深色模式、系统托盘全局快捷键等功能。

## 三、功能需求

### P0：核心功能

#### 3.1 自定义无边框窗口

| 项目 | 说明 |
|------|------|
| 窗口样式 | 去除系统原生标题栏，使用自定义标题栏 |
| 标题栏内容 | 应用图标 + 应用名称 + 搜索框 + 统计入口 + 最小化/最大化/关闭按钮 |
| 窗口操作 | 标题栏区域拖拽可移动窗口；双击标题栏可最大化/还原 |
| 窗口缩放 | 鼠标移至窗口四边/四角可拖拽调整大小（需自行实现 `nativeEvent` 或边缘检测） |
| 默认尺寸 | 900 × 600，最小尺寸 700 × 450 |
| 窗口状态持久化 | 关闭时保存窗口位置和尺寸，下次启动恢复（`QSettings`） |

#### 3.2 便签数据管理

**数据结构：**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 唯一标识（UUID） |
| title | String | 便签标题 |
| content | String | 便签正文（支持富文本 HTML） |
| category | String | 分类标识 |
| color | String | 背景色：黄 `#FFEAA7`、青 `#81ECEC`、灰 `#DFE6E9`、红 `#FAB1A0`、紫 `#A29BFE` |
| pinned | Bool | 是否置顶，默认 false |
| sortOrder | Int | 手动排序序号 |
| createdAt | DateTime | 创建时间 |
| modifiedAt | DateTime | 最后修改时间 |

**存储方式：** 本地 JSON 文件（`~/.stickynote/notes.json`），启动时加载，变更时自动保存。需处理文件读写异常（文件损坏、权限不足等），写入时先写临时文件再原子替换。

#### 3.3 便签列表展示

以卡片形式展示便签，使用 Qt Model/View 架构（`QAbstractListModel` + `QStyledItemDelegate`）。

**单张卡片布局：**

```
┌────────────────────────────┐
│ 📌 周会纪要                 │  ← 标题（加粗）+ 置顶图标
│ 讨论了Q2的规划方向，重点是... │  ← 内容预览（灰色，截取前50字）
│ 🏷️ 工作          今天 10:30 │  ← 分类标签 + 修改时间
└────────────────────────────┘
```

| 项目 | 规格 |
|------|------|
| 卡片圆角 | 8px |
| 标题字号 | 14px bold |
| 正文字号 | 12px，灰色，最多显示 50 字（需过滤 HTML 标签后截取） |
| 时间字号 | 10px，灰色 |
| 置顶标记 | 标题前显示 📌 图标 |
| 卡片背景色 | 跟随便签 color 字段 |

**排序规则：** 置顶便签优先，同级按 `sortOrder` 排序，`sortOrder` 相同按修改时间倒序。

#### 3.4 新建 / 编辑便签

通过模态弹窗进行新建和编辑操作，弹窗复用同一组件。

**弹窗布局：**

```
┌──────────────────────────────────────┐
│         编辑便签                 ✕    │
├──────────────────────────────────────┤
│ 标题：[____________________________] │
│                                      │
│ 内容：  B  I  U  [文字色▼] [列表]    │  ← 富文本工具栏
│ ┌──────────────────────────────────┐ │
│ │      （QTextEdit 富文本编辑器）    │ │
│ └──────────────────────────────────┘ │
│                                      │
│ 分类：[工作 ▼]                       │
│                                      │
│ 颜色：  ● ● ● ● ●                   │
│        黄 青 灰 红 紫                 │
│                                      │
│ 字数：128 字                          │
│                                      │
│         [ 取消 ]  [ 保存 ]           │
└──────────────────────────────────────┘
```

| 交互 | 说明 |
|------|------|
| 新建入口 | 点击"+ 新建便签"按钮 |
| 编辑入口 | 双击便签卡片 |
| 保存快捷键 | `Ctrl+S` |
| 未保存拦截 | 编辑内容未保存时关闭弹窗，弹出二次确认框 |
| 富文本工具栏 | 加粗、斜体、下划线、文字颜色、无序列表，通过 `QTextEdit` + `QTextCharFormat` 实现 |
| 实时字数统计 | 编辑区下方实时显示当前字数 |

#### 3.5 删除便签

| 交互 | 说明 |
|------|------|
| 删除入口 | 便签卡片右键菜单 → 删除 |
| 二次确认 | 弹出确认对话框，确认后删除 |
| 批量删除 | `Ctrl+Click` 多选便签后可批量删除 |

#### 3.6 分类管理

| 交互 | 说明 |
|------|------|
| 内置分类 | 全部 / 工作 / 生活 / 学习（不可删除） |
| 自定义分类 | 侧边栏底部"+ 自定义"按钮，弹出输入框新建分类 |
| 分类计数 | 每个分类名称后显示该分类下的便签数量 |
| 删除分类 | 右键自定义分类可删除，删除时该分类下便签归入"未分类" |
| 筛选方式 | 点击侧边栏分类项，通过 `QSortFilterProxyModel` 代理过滤 |
| 空状态 | 当前分类无便签时显示占位提示 |

#### 3.7 搜索

| 交互 | 说明 |
|------|------|
| 搜索入口 | 标题栏内嵌搜索框 |
| 搜索范围 | 匹配便签标题和正文内容 |
| 搜索方式 | 输入即搜，实时过滤（增加 300ms 防抖） |
| 高亮 | 搜索结果卡片中，匹配关键词高亮显示 |
| 无结果 | 显示空状态占位提示 |

#### 3.8 右键菜单

便签卡片右键弹出上下文菜单：

| 菜单项 | 说明 |
|------|------|
| 编辑 | 打开编辑弹窗 |
| 删除 | 二次确认后删除 |
| 置顶 / 取消置顶 | 切换置顶状态 |
| 更改分类 | 子菜单：动态读取所有分类 |
| 更改颜色 | 子菜单：5 种颜色选项 |
| 独立窗口打开 | 将该便签以独立小窗口展示（见 3.16） |

#### 3.9 颜色选择器

自定义控件，5 个彩色圆形按钮横排排列，点击选中时显示对勾标记。通过 `paintEvent` 手动绘制，支持键盘左右方向键切换选中。

### P1：体验优化

#### 3.10 自定义图标按钮（IconButton）

三态图标按钮（normal / hover / pressed），通过 `paintEvent` 自绘，支持 `setEnabled(false)` 禁用态灰显。用于标题栏和工具栏按钮。

#### 3.11 Toast 轻提示

| 项目 | 说明 |
|------|------|
| 触发场景 | 保存成功、删除成功、导入完成等操作反馈 |
| 展示位置 | 窗口底部居中 |
| 展示时长 | 1.5 秒后自动消失 |
| 动画效果 | 透明度渐入渐出（`QPropertyAnimation`） |
| 多条排队 | 多条 Toast 同时触发时，依次排队展示，不重叠 |

#### 3.12 卡片 Hover 动画

鼠标悬浮时卡片轻微上浮 + 阴影加深，使用 `QGraphicsDropShadowEffect` + `QPropertyAnimation` 实现。

#### 3.13 系统托盘与全局快捷键

| 交互 | 说明 |
|------|------|
| 关闭行为 | 点击关闭按钮隐藏到托盘，不退出应用 |
| 恢复窗口 | 双击托盘图标 |
| 托盘右键菜单 | 显示主窗口 / 快速新建 / 退出 |
| 全局快捷键 | `Ctrl+Alt+N` 全局唤起快速新建便签窗口（需注册系统热键） |

#### 3.14 卡片拖拽排序

| 项目 | 说明 |
|------|------|
| 交互方式 | 鼠标长按便签卡片后拖拽，改变排列顺序 |
| 视觉反馈 | 拖拽时显示卡片半透明跟随效果，目标位置显示插入线指示 |
| 数据同步 | 拖拽完成后更新各便签 `sortOrder` 字段，持久化到 JSON |
| 实现要求 | 在 `QListView` 上重写 `startDrag`、`dragEnterEvent`、`dropEvent` 等 |

#### 3.15 撤销 / 重做操作栈

| 项目 | 说明 |
|------|------|
| 支持操作 | 新建、删除、编辑、置顶、移动分类、排序变更 |
| 栈深度 | 最近 20 步 |
| 快捷键 | `Ctrl+Z` 撤销，`Ctrl+Y` 重做 |
| 工具栏入口 | 底部工具栏撤销/重做按钮，无可撤销操作时按钮禁用态 |
| 实现方式 | Command 模式：定义 `UndoCommand` 基类，每种操作实现 `execute()` / `undo()` |

### P2：进阶功能

#### 3.16 独立便签小窗口

| 项目 | 说明 |
|------|------|
| 入口 | 右键菜单"独立窗口打开" |
| 窗口样式 | 无边框小窗口（300×200），显示单条便签内容，可直接编辑 |
| 置顶 | 独立窗口默认置顶在所有窗口之上（`Qt::WindowStaysOnTopHint`） |
| 数据同步 | 小窗口内编辑后，主窗口列表实时同步更新（信号/槽） |
| 多窗口 | 可同时打开多个独立便签窗口 |

#### 3.17 深色 / 浅色模式切换

| 项目 | 说明 |
|------|------|
| 切换入口 | 底栏切换按钮 |
| 实现方式 | 两套 QSS 样式文件（`light.qss` / `dark.qss`），动态加载 |
| 切换动画 | 切换时整体透明度渐变过渡，不闪烁 |
| 要求 | 所有自定义控件（包括 Delegate 绘制的卡片、独立小窗口）在两种模式下均显示正常 |
| 持久化 | 记住用户上次选择的主题 |

#### 3.18 便签导入 / 导出

| 操作 | 说明 |
|------|------|
| 导出 | 全部便签序列化为 JSON 文件，通过文件选择对话框保存 |
| 导入 | 从 JSON 文件读取便签，合并到当前列表（ID 去重） |
| 导入校验 | 校验 JSON 格式合法性，非法数据跳过并 Toast 提示 |

#### 3.19 便签统计面板

| 项目 | 说明 |
|------|------|
| 入口 | 标题栏"统计"按钮，弹出统计弹窗 |
| 统计内容 | 各分类便签数量柱状图（`QPainter` 手动绘制）、总便签数、本周新增数、最近编辑时间 |
| 图表绘制 | 不使用第三方库，用 `QPainter` 在 `QWidget::paintEvent` 中手动绘制柱状图 |
